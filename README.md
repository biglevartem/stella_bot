# Felo Assistant Bot

Консультант по продуктам Felo, который работает в виде веб-страницы и Telegram-бота. В основе лежит единая база знаний о товарах и условиях доставки.

## Возможности

- Подбор продуктов по целям (энергия, концентрация, сон, иммунитет).
- Просмотр подробной информации о каждом продукте: состав, преимущества, статус наличия, рекомендации по применению.
- Ответы на типовые вопросы о доставке, оплате и возвратах.
- Быстрые подсказки и сценарии для клиента в веб-версии.
- Дублирование той же логики в Telegram-боте.

## Структура проекта

```
├── bot/
│   ├── knowledge.py       # Общая логика и загрузка ассортимента
│   └── telegram_bot.py    # Телеграм-бот на python-telegram-bot
├── data/
│   └── products.json      # Описание ассортимента Felo и правил
├── web/
│   ├── index.html         # Интерфейс консультанта
│   ├── script.js          # Логика чат-бота в браузере
│   └── styles.css         # Оформление
├── requirements.txt       # Зависимости для Telegram-бота
└── README.md
```

## Запуск веб-версии

1. Откройте терминал в корне проекта.
2. Запустите любой статический сервер, например встроенный Python:
   ```bash
   python -m http.server 8000
   ```
3. Перейдите в браузере на <http://localhost:8000/web/>.
4. Общайтесь с ботом: используйте быстрые кнопки или введите свою задачу.

> Примечание: данные подгружаются из `data/products.json`. При изменении файла просто обновите страницу.

## Запуск Telegram-бота

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте бота в [@BotFather](https://t.me/BotFather) и получите токен.
3. Экспортируйте токен как переменную окружения:
   ```bash
   export TELEGRAM_BOT_TOKEN="ваш_токен"
   ```
4. Запустите бота:
   ```bash
   python -m bot.telegram_bot
   ```
5. В Telegram перейдите к своему боту и начните диалог командой `/start`.

Telegram-версия использует тот же набор правил и ассортимент, что и веб-страница.

## Кастомизация ассортимента

- Все данные и ответы задаются в `data/products.json`.
- Можно менять описание продуктов, условия доставки или платежи — изменения автоматически подхватываются обеими версиями бота.
- При необходимости расширить логику, редактируйте `bot/knowledge.py` и `web/script.js`.
